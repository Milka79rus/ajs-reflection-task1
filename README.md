# Object, Reflection and Proxy — Task 1: `for...in`

[![Check Code](https://github.com/Milka79rus/ajs-reflection-task1/actions/workflows/run.yml/badge.svg)](https://github.com/Milka79rus/ajs-reflection-task1/actions/workflows/run.yml)

## Описание
В данном проекте реализована функция `orderByProps`, которая сортирует свойства объекта по заданному алгоритму:
1. Сначала идут свойства, указанные в специальном массиве порядка.
2. Все остальные свойства объекта сортируются в алфавитном порядке по названию ключа.

Проект выполнен в рамках учебного курса по продвинутому JavaScript.

## Технологии и инструменты
* **JavaScript (ES6+)**
* **Webpack** — сборка проекта.
* **Jest** — модульное тестирование.
* **ESLint** — проверка стиля кода.

## Как это работает
Функция принимает два аргумента:
- `obj`: исходный объект.
- `sortOrder`: массив строк с именами ключей, определяющий приоритет вывода.

## Тестирование
Проект покрыт тестами на 100% (Statements, Branches, Functions, Lines).
Протестированы следующие сценарии:

* Сортировка по заданному порядку и алфавиту.

* Работа при пустом массиве сортировки.

* Игнорирование в массиве сортировки ключей, которых нет в объекте.

## Установка и запуск
1. Клонируйте репозиторий.

2. Установите зависимости: `npm install`.

3. Запустите тесты: `npm test`.

4. Для генерации отчета о покрытии: `npm run coverage`.
